# Challenge ONE Java: Conversor de Monedas 💱

¡Bienvenido a mi Conversor de Monedas! Este proyecto fue desarrollado como parte del programa **Oracle Next Education** en colaboración con **Alura**.

## 🚀 Funcionalidades
El programa permite realizar conversiones en tiempo real entre las siguientes monedas:
* Dólar (USD) ↔ Peso Argentino (ARS)
* Dólar (USD) ↔ Real Brasileño (BRL)
* Dólar (USD) ↔ Peso Colombiano (COP)

## 🛠️ Tecnologías Utilizadas
* **Java 17** (LTS)
* **Biblioteca Gson** (v2.13.2) para la manipulación de JSON.
* **ExchangeRate-API** para obtener tasas de cambio en vivo.
* **HttpClient** para las solicitudes asíncronas.
